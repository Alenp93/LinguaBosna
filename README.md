# LinguaBosna

**LinguaBosna** ist eine kostenlose Lernwebseite für die bosnische Sprache,
primär für deutschsprachige Lernende, die Diaspora sowie Kinder, Jugendliche
und Erwachsene aller Sprachniveaus (A1–C2).

🌐 **Website:** [www.linguabosna.com](https://www.linguabosna.com)

---

## Über das Projekt

LinguaBosna bietet strukturierte Grammatikerklärungen, Vokabelübungen und
interaktive Inhalte — ohne Registrierung, ohne Kosten, ohne Werbung.

Die Webseite ist als reines Frontend-Projekt umgesetzt (HTML, CSS, JavaScript)
und wird über GitHub Pages gehostet.

**Aktueller Stand:**
- ✅ Startseite, Header, Footer
- ✅ Grammatikbereich (A1, laufend erweitert)
- ✅ Vokabelbereich
- 🔄 Weitere Grammatikkategorien in Entwicklung
- 🔜 Kulturteil und Blog (geplant)
